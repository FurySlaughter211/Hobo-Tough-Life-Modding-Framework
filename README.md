# Hobo: Tough Life Modding Framework 

This is a framework for Hobo: Tough Life that allows for the creation and loading of custom mods. It is the first project aimed at making the game fully moddable by providing a foundation for adding new content.

The framework is built on BepInEx and uses a JSON-based system to make adding content as easy as possible. Instead of writing code, you can define your items, models, and recipes in simple text files, and the framework will handle injecting them into the game.

**Note: This project is in its very early stages. Expect bugs, and please back up your save files before using any mods.**
